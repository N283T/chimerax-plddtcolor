Command: plddt

Usage:
plddt [ atoms ]

The plddt command colors atoms based on their pLDDT (predicted Local Distance Difference Test) values. pLDDT values are typically stored as B-factor values in protein structure files and provide a measure of prediction confidence for each residue. This command automatically applies a color scheme to visualize the prediction accuracy across the protein structure.

Arguments:

  • atoms - Optional atom specification. If not provided, all atoms in all models will be colored. If specified, only the selected atoms will be colored based on their pLDDT values.

Color Scheme:

The command applies the following color scheme based on pLDDT values:

  • Very High (90-100): Blue (#0053D6) - Very high confidence
  • High (70-89): Light Blue (#65CBE3) - High confidence
  • Medium (50-69): Yellow (#FFDB13) - Medium confidence
  • Low (0-49): Orange (#FF7D45) - Low confidence

Output:

The command provides detailed statistics for each processed model, including:

  • Minimum, maximum, mean, and median pLDDT values
  • Standard deviation of pLDDT values
  • Total number of atoms processed
  • Number of atoms colored at each threshold

Statistics are displayed in both the log and as an HTML table for easy reading.

Examples:

plddt
Color all atoms in all models based on their pLDDT values
plddt sel
Color only the currently selected atoms based on their pLDDT values
plddt #1
Color all atoms in model #1 based on their pLDDT values

Notes:

  • pLDDT values must be stored as B-factor values in the structure file
  • Ribbon colors are automatically updated to match the atom colors
  • The command processes multiple models simultaneously if present
  • Statistical information is provided for each model separately
