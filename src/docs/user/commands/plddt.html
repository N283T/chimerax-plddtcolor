Command: color plddt

Usage:
color plddt  spec [ high_cutoff high_cutoff ] [ high_color colorname ] [ mid_high_cutoff mid_high_cutoff ] [ mid_high_color colorname ] [ mid_low_cutoff mid_low_cutoff ] [ mid_low_color colorname ] [ low_cutoff low_cutoff ] [ low_color colorname ] [ attribute attribute ]

The color plddt command colors atoms and ribbons based on pLDDT values or B-factor values. This is particularly useful for visualizing AlphaFold2 prediction results or other structure quality metrics.

By default, the command uses B-factor values and applies a color scheme with the following thresholds:

  • ≥ 100: Blue (#0053D6) - Very high confidence
  • ≥ 90: Light blue (#65CBF3) - High confidence
  • ≥ 70: Yellow (#FFDB13) - Medium confidence
  • ≥ 50: Orange (#FF7D45) - Low confidence
  • < 50: Orange (#FF7D45) - Very low confidence

The high_cutoff, mid_high_cutoff, mid_low_cutoff, and low_cutoff options allow you to customize these thresholds. The high_color, mid_high_color, mid_low_color, and low_color options allow you to customize the colors. The attribute option allows you to specify which atomic attribute to use for coloring (default is "bfactor").

If no atoms are specified, all atoms in the current session will be colored.
